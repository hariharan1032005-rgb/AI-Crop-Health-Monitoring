# 🌱 AI Crop Health Monitoring System

An AI-powered system that detects crop diseases from leaf images and predicts pest risk using weather data.

## Features
- Leaf disease detection using AI
- Weather-based pest risk prediction
- Simple web interface
- Image upload for crop analysis

## Technologies Used
- Python
- Flask
- Machine Learning
- HTML / CSS
- Weather API
